# Qumi (統合思考OS / 自己拡張システム)

![System Status](https://img.shields.io/badge/System-Integrated_Thinking_OS-00FF99?style=for-the-badge)
![Philosophy](https://img.shields.io/badge/Philosophy-Exploration_of_the_Unknown-B026FF?style=for-the-badge)

## 📌 概要 (Overview)
**Qumi** は、単なるツールやアプリケーションではなく、自分自身の「未知への探求心」と「思考プロセス」をシステムとして拡張・委譲するための**統合思考OS（自己拡張システム）**です。
日々の学習、気づき、アイディア（外部脳としての魂の出力）を、抽象度の高いレイヤーからCLI/GUIへと流し込み、AIエージェントに自律的に構造化させる実験的なアーキテクチャを持ちます。

## 🧠 設計の哲学 (Philosophy)
このリポジトリは、就職活動における単なるポートフォリオではありません。
常に新しい技術を吸収し、社会・システムに対して「とにかく聞き、学ぶ態度（初心を忘れない態度）」を証明し続けるための**生きた実験場**です。

*   **GUIレイヤー (`/ui`)**: Apple/iOSライクな圧倒的に美しくモダンなGlassmorphism（すりガラス）UI。物理演算によるバネのような心地よいインタラクションを取り入れています。
*   **CLI・エージェントレイヤー (`/agents`)**: *Soul Delegation Engine*. 視覚的なノードベースではなく、`browser-use` などの最先端のCLI・LLMエージェントを採用。抽象度のレイヤーを変えながら、本質的な「思考の構造化問題」を解いています。
*   **外部脳・ログ (`/docs`)**: シナプスのように繋がり合う思考のプロセス、アーキテクチャの変遷、技術的気付きの記録（例: n8nとClaude Codeのレイヤー移行による気づきなど）。

## 📁 ディレクトリ構造 (Architecture)
```
Qumi/
 ├── .github/          # 開発を補助するAIエージェント（Copilot等）への絶対規則・スキルファイル
 ├── agents/           # ブラウザ自動操作・JSON化などの自律型LLMエージェント（Python）
 ├── docs/             # 外部脳（システム構想・戦略・アーキテクチャのログ）
 └── ui/               # コントローラーUIのソースコード（React Native Web / Reanimated）
```

## 🚀 今後の展望 (Roadmap)
このパブリックリポジトリは、就職活動や短期的な目標の枠に収まらず、今後のキャリア全体（大規模インフラの思考、設計開発など）に活きる「自己設計のプラットフォーム」として成長し続けます。ツールが変わろうとも解決する問題のコアは同じであるという気付きを胸に、常にアップデートを続けていきます。
