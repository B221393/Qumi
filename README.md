# Qumi (統合思考OS / 自己拡張システム)

![System Status](https://img.shields.io/badge/System-Integrated_Thinking_OS-00FF99?style=for-the-badge)
![Philosophy](https://img.shields.io/badge/Philosophy-Exploration_of_the_Unknown-B026FF?style=for-the-badge)

## 📌 概要 (Overview)
**Qumi** は、単なるツールやアプリケーションではなく、自分自身の「未知への探求心」と「思考プロセス」をシステムとして拡張・委譲するための**統合思考OS（自己拡張システム）**です。
日々の学習、気づき、アイディア（外部脳としての魂の出力）を、抽象度の高いレイヤーからCLI/GUIへと流し込み、AIエージェントに自律的に構造化させる実験的なアーキテクチャを持ちます。

## 🔒 公開ポリシー・精査基準 (Public Disclosure Policy)
本リポジトリは、オープンな技術共有およびアーキテクチャ設計の証明を目的とし、**個人情報や機密性の高い就職活動のログ等が含まれないよう、システム的に公開基準を厳格に精査（Sanitize）して公開しています。**

*   🟢 **【公開しているもの (Public Assets)】**
    *   **フロントエンドUX/UIの基盤実装**：iPhone/iOSライクなGlassmorphism（すりガラス）や物理演算バウンスアニメーションの検証コード（`ui/App.tsx`）
    *   **エージェント連携のプロトタイプ**：LLMとブラウザ自動操作（browser-use）を用いた、抽象的な思考の構造化スクリプト（`agents/soul_delegation_agent.py`）
    *   **アーキテクチャ設計の知見**：n8nからCLI環境への移行に関するレイヤー論など、純粋な技術的・哲学的な学び（`docs/`）

*   🔴 **【非公開としているもの (Private/Confidential)】**
    *   個人的な面接戦略、企業分析、および就職活動に係る具体的なログ（これらはすべて別途プライベートな環境で管理しています）
    *   個人を特定するデータや、バックエンドで稼働する各種APIキー（`.env`等の機密設定）

この情報管理と公開基準の明確な分離は、大規模システムやインフラ技術の開発において不可欠な**「セキュリティ・ガバナンス意識」**の表れでもあります。

## 🧠 設計の哲学 (Philosophy)
このリポジトリは、常に新しい技術を吸収し、社会・システムに対して「とにかく聞き、学ぶ態度（初心を忘れない態度）」を証明し続けるための**生きた実験場**です。
「ツール（手段）」への過度な依存を脱却し、その底にある「課題解決のコア」を見極めるための思考の結晶が、この `Qumi` プロジェクトの根幹にあります。

## 📁 ディレクトリ構造 (Architecture)
```
Qumi/
 ├── .github/          # 開発を補助するAIエージェントへの絶対規則・スキルファイル（精査済み）
 ├── agents/           # ブラウザ自動操作・JSON化などの自律型LLMエージェント
 ├── docs/             # 外部脳（システム構想・戦略・技術的知見のログ）
 └── ui/               # コントローラーUIのソースコード（React Native Web / Reanimated）
```

## 🚀 今後の展望 (Roadmap)
長期的なキャリア（大規模インフラの思考、設計開発など）に活きる「自己設計のプラットフォーム」として成長し続けます。ツールが変わろうとも解決する問題のコアは同じであるという気付きを胸に、常にアップデートを続けていきます。
