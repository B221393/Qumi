# Qumi — 就活管理ダッシュボード

## 概要

**Qumi** は、12社の就職活動（就活）の進捗をブラウザ上で一元管理するシンプルな静的ウェブアプリです。  
バックエンド不要で、すべての状態は `localStorage` に保存されるため、インターネット接続がなくても動作します。

## 公開ファイル

| ファイル | 説明 |
|---|---|
| `index.html` | メインページ（ダッシュボード・モーダル） |
| `style.css`  | グラスモーフィズム UI スタイル |
| `script.js`  | 企業データ管理・画面描画ロジック |

## 主な機能

- **12社の就活進捗管理** — 各企業カードをクリックしてステータスとメモを更新
- **ステータス管理** — 未応募 / 応募済み / 書類選考 / 面接 / 最終面接 / 内定 / お祈り
- **サマリーバー** — 応募済み・選考中・面接・内定の件数をリアルタイム表示
- **ローカル保存** — `localStorage` を使ってブラウザ内にデータを永続化（サーバー不要）
- **レスポンシブデザイン** — モバイル・タブレット・デスクトップ対応

## 使い方

1. `index.html` をブラウザで開く（またはウェブサーバーで配信する）
2. 各企業カードをクリックするとステータス編集モーダルが開く
3. ステータスとメモを編集して「保存」ボタンを押す
4. ダッシュボード上部のサマリーバーが自動更新される

## 技術スタック

- HTML5 / CSS3（グラスモーフィズム、CSS カスタムプロパティ）
- Vanilla JavaScript（ES6+、localStorage API）
- Google Fonts（Outfit / Noto Sans JP）
